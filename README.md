## Featured Projects

### [Cybersecurity Audit Automation Platform for SMEs](https://github.com/KanikaIm-erb/Cybersecurity-Audit-Platform-for-SMEs)

`Python` `Flask` `React.js` `Vite` `Nmap` `OWASP ZAP` `SQLite` `Docker`

SMEs make up over 43% of global cybercrime targets, yet most don't have the budget or headcount for regular security audits. This platform automates the full audit pipeline — scanning, analysis, compliance mapping, and reporting — in one unified, open-source tool built with non-technical users in mind.

What it covers:

`Network scanning` — host discovery, port enumeration, and service fingerprinting via Nmap and Masscan

`Web application scanning` — OWASP Top 10 testing including SQLi, XSS, misconfigurations, and SSL/TLS weaknesses via OWASP ZAP and Nikto

`Configuration auditing` — detects hardcoded secrets, exposed `.env` files, and insecure server settings via TruffleHog, Lynis, and Checkov

`Database auditing` — checks for exposed ports, default credentials, and excessive privileges

`Compliance mapping` — findings automatically mapped to ISO 27001, NIST CSF, and PCI DSS controls

`Reporting dashboard` — risk-scored, prioritized findings with remediation guidance, exportable as PDF, HTML, JSON, or CSV

The goal was to translate raw tool output into something a founder or small IT team could actually read and use — with clear risk scores, business-impact explanations, and guided next steps.

### [Real-Time AI-Based Vishing Scam Detector](https://vishing-detector-kanika-imerb.streamlit.app/)

`Python` `Whisper` `NLP` `TF-IDF` `Logistic Regression` `Scikit-learn` `Streamlit`

Vishing (voice phishing) is one of the fastest-growing social engineering threats, and it hits harder in regions where most detection tools only support English. This tool analyzes live phone call audio and flags potential scam calls in real time, with full support for both Thai and English.

How it works:

`Step 1` — Captures and transcribes live audio using OpenAI Whisper (speech-to-text)

`Step 2` — Runs the transcript through a Logistic Regression classifier trained on character-level TF-IDF features

`Step 3` — Flags social engineering patterns: urgency cues, impersonation language, pressure tactics, financial lure phrases

`Result` — 95.4% classification accuracy on a multilingual English and Thai test set
